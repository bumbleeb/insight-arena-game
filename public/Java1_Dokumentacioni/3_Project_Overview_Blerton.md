# Project Overview - BrainQuest

**Studenti:** Blerton  
**Projekti:** BrainQuest Quiz App

---

## Përmbledhje

BrainQuest është aplikacion quiz edukativ për studentë 13-18 vjeç me 4 kategori: Shkencë, Histori, Letërsi, Mendim Kritik.

---

## Karakteristikat

- 4 kategori × 15 pyetje
- 3 nivele vështirësie
- Shumëgjuhësi (EN/SQ)
- Score tracking
- Feedback vizual
- Animated backgrounds

---

## Stack Teknologjik

**Frontend:**
- React 18 + TypeScript
- Vite
- Tailwind CSS
- React Router
- TanStack Query

**Tools:**
- ESLint
- Lucide React
