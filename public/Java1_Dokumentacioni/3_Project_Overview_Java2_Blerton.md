## Java 2 - Design System

**Përshkrimi:**
Krijimi i sistemit të dizajnit në src/index.css me CSS variables, gradiente, hije dhe animacione.

**Komponente të Krijuara:**

1. **CSS Variables**
   - Ngjyra bazë (primary, secondary, accent, success)
   - Ngjyra të foreground për text
   - Ngjyra të card dhe popover
   - Border, input, dhe ring colors

2. **Gradientet Custom**
   - gradient-primary: Vjollcë → Kaltër
   - gradient-secondary: Kaltër → Cyan
   - gradient-accent: Rozë → Vjollcë
   - gradient-success: Jeshile → Teal

3. **Hije Custom**
   - shadow-glow: Shkëlqim për primary elements
   - shadow-card: Hije për card components

4. **Animacione CSS**
   - fade-in: Shfaqje me opacity dhe translateY
   - scale-in: Shfaqje me scale dhe opacity
   - slide-up: Lëvizje nga poshtë
   - accordion-down/up: Expand dhe collapse

5. **Dark Mode**
   - Variables të redefinuara për dark mode
   - Automatic switching me class="dark"
   - Contrast optimal për readability

---

**Karakteristikat:**
- Te gjitha ngjyrat në format HSL
- Semantic tokens për colors
- Smooth transitions
- Accessible color contrast
- Reusable design system
